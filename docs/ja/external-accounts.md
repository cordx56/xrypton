# 外部アカウント連携

Xryptonは外部のアカウントとの連携として、Xryptonユーザがその外部アカウントの所有者であることを証明するための機能を有しています。

## Bluesky (AT Protocol) 連携

SNSのBluesky（で用いられているAT Protocol）に公開鍵情報を投稿し、この投稿のメタデータに署名することで、XryptonユーザがそのAT Protocolアカウントを所有していることを証明することができます。

詳しくは[AT Protocol連携](/docs/ja/atproto)をお読みください。

## X連携

SNSのXに公開鍵情報を投稿し、この投稿のメタデータに署名することで、XryptonユーザがそのXアカウントを所有していることを証明することができます。

連携はプロフィール画面からできます。
プロフィール画面で連携を開始し、公開鍵情報を含むメッセージを投稿、そのURLを取得してXryptonの入力欄に貼り付けてください。
この情報について署名し、サーバに保存することで、Xアカウントの所有を証明できます。
