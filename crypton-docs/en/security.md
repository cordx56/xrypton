# Security

Crypton places a strong emphasis on security.

## Signing and Encryption

Most data on Crypton is signed, and data sent as messages is encrypted.
The following table shows which data is signed or encrypted.

| Data             | Security           |
|------------------|--------------------|
| Profile image    | Signed             |
| Name             | Signed             |
| Status           | Signed             |
| Bio              | Signed             |
| Channel name     | None               |
| Thread name      | None               |
| Updated at       | None               |
| Message          | Signed / Encrypted |
| Attached image   | Signed / Encrypted |
| Attached file    | Signed / Encrypted |
