# Cryptonドキュメント

Cryptonドキュメントへようこそ。Cryptonは、暗号技術に基づいて構築されたエンドツーエンド暗号化メッセージングプラットフォームです。

## 特徴

- **エンドツーエンド暗号化** : すべてのメッセージはPGP鍵を使って署名、暗号化され、保護されます。
- **オープンソース** : コードはすべて公開されていて、誰でも自由に利用できます。
- **分散システム** : 特定の運営者に依存せず、好きなサーバを選択できます。
