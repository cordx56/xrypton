# セキュリティ

Cryptonはセキュリティを重視しています。

## 署名と暗号化

Crypton上のほとんどのデータは署名され、メッセージとして送られたデータは暗号化されます。
以下にどのようなデータが署名または暗号化されるかを示した表を掲載します。

| データ           | セキュリティ  |
|------------------|---------------|
| プロフィール画像 | 署名          |
| 名前             | 署名          |
| ステータス       | 署名          |
| 自己紹介         | 署名          |
| チャンネル名     | なし          |
| スレッド名       | なし          |
| 更新日時         | なし          |
| メッセージ       | 署名 / 暗号化 |
| 添付画像         | 署名 / 暗号化 |
| 添付ファイル     | 署名 / 暗号化 |
