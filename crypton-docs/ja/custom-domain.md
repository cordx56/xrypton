# カスタムドメインを設定する

Cryptonには分散サーバを設置する以外にも、自分のドメインを使う方法があります。

## DNSにTXTレコードを追加する

どこかのサーバでカスタムドメインを使うには、まず自身のドメインのTXTレコードに値を追加する必要があります。

まず、あなたの所属したいサーバ `[host-domain]` と使いたいユーザID、 `[ID]@[your-domain]` を決めます。
決まったら、あなたのドメイン `[your-domain]` の `_crypton.[your-domain]` に、以下のようなTXTレコードを追加します。

`"user=[ID]@[host-domain]"`

重要なポイントとして、TXTレコードの値には `"` も含めてください。

## PGP鍵を生成して登録する

ここまでできたら、PGP鍵を生成して登録しましょう。

まず、あなたの所属したいサーバ、 `[host-domain]` にアクセスし、登録画面を開きます。
鍵生成画面の「ユーザID」に、 `[ID]@[your-domain]` を入力します。

以上！簡単でしょ？

## トラブルシューティング

- 登録画面で「DNS mapping not found for this domain」と表示される
  - TXTレコードの形式が間違っています。一回間違えると、間違えた結果が1時間程度キャッシュされます。設定を確認して、時間をおいて再チャレンジしてみましょう。
